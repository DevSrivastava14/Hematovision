# HematoVision: Blood Cell Classification using AI

## Overview
This project classifies blood cell images into:
- Eosinophil
- Lymphocyte
- Monocyte
- Neutrophil

## Technologies Used
- Python
- TensorFlow / Keras
- Flask
- HTML / CSS

## Features
- Upload blood cell image
- Predict cell type using trained model
- Web-based interface

## Model
- Transfer Learning using MobileNetV2

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   python app.py

3. Open browser:
   http://127.0.0.1:5000
